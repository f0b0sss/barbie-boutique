#### **Project in developing process**

## Multi locale Doll Clothes Eshop


### Features:
* Multi locale
* Saving data to Postgresql
* User Registration with data validation
* Email verification
* Password restore
* Sorting products by categories
* Creating order
* Confirming Order by admin

### Technologies:
* Java
* Spring boot
* JPA
* Spring security
* Thymeleaf
* Flyway
* Spring mail
